# 课时达人

一个适合手机使用的课时记录小网页。

## 已有功能

- 新增上课记录
- 记录上课具体时间点
- 直接填写课时数
- 选择课时类型：阶梯课时 / 分成课时
- 录课时可直接选择学生
- 可快速新增学生
- 支持批量导入学生名单
- 学生可标记为在读 / 已停课
- 学生管理默认折叠，按需展开
- 首页默认显示：本月课时、本月阶梯课时、本月分成课时
- 总统计默认折叠，按需展开
- 按月份、学生、课时类型筛选
- 按学生/课程/备注搜索
- 导出 CSV
- 数据保存在浏览器本地

## 说明

- 批量导入时支持一行一个名字，也支持逗号、顿号分隔
- 建议对停课学生使用“停课”而不是删除，这样历史记录不会乱
- 目前数据保存在当前浏览器本地，换手机或换浏览器不会自动同步
