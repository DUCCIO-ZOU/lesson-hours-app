# 课时达人

一个适合手机使用的课时记录小网页。

## 已有功能

- 新增上课记录只需：日期、学生、课时类型、课时数、备注
- 学生支持一次多选，保存后自动生成多条记录
- 课时数改为按钮选择：1课时 / 1.5课时 / 2课时
- 保存后会显示完成提醒
- 录课时可直接选择学生
- 可快速新增学生
- 支持批量导入学生名单
- 学生可标记为在读 / 已停课
- 学生管理默认折叠，按需展开
- 本月统计默认折叠，按需展开
- 总统计默认折叠，按需展开
- 记录列表默认折叠，按需展开
- 按月份、学生、课时类型筛选
- 按学生/备注搜索
- 导出 CSV
- 数据保存在浏览器本地

## 说明

- 旧版记录中的“时间/课程”字段会自动忽略，不影响现有课时数据
- 批量导入时支持一行一个名字，也支持逗号、顿号分隔
- 旧版“分成课时”会自动迁移成“陪练课时”
- 建议对停课学生使用“停课”而不是删除，这样历史记录不会乱
- 目前数据保存在当前浏览器本地，换手机或换浏览器不会自动同步
