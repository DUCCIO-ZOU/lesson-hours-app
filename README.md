# 课时达人（云端同步版）

现在这个版本已经接入 Supabase 云端数据库。

## 已有功能

- 手机和电脑共用同一份学生名单
- 手机和电脑共用同一份课时记录
- 新增上课记录只需：日期、学生、课时类型、课时数、备注
- 学生支持一次多选，保存后自动生成多条记录
- 课时数改为按钮选择：1课时 / 1.5课时 / 2课时
- 保存后会显示完成提醒
- 可快速新增学生
- 支持批量导入学生名单
- 学生可标记为在读 / 已停课
- 学生管理、本月统计、总统计、记录列表都支持折叠
- 按月份、学生、课时类型筛选
- 按学生/备注搜索
- 导出 CSV

## 说明

- 数据现在保存在 Supabase 云端，不再只存在浏览器本地
- 电脑和手机刷新后都会看到同一份最新数据
- 旧版“分成课时”会自动视为“陪练课时”
- 建议对停课学生使用“停课”而不是删除，这样历史记录不会乱
