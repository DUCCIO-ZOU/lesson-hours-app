# 课时达人

一个适合手机使用的课时记录小网页。

## 已有功能

- 新增上课记录
- 记录上课具体时间点
- 直接填写课时数
- 选择课时类型：阶梯课时 / 分成课时
- 查看总记录数、总课时、本月课时
- 单独统计阶梯课时、分成课时
- 单独统计本月阶梯课时、本月分成课时
- 按月份筛选
- 按课程/备注搜索
- 按课时类型筛选
- 导出 CSV
- 数据保存在浏览器本地
- 适合个人单独使用，不需要老师字段

## 怎么打开

### 电脑上
直接双击 `index.html` 即可。

### 手机上
推荐部署到 GitHub Pages，手机浏览器打开网址就能用。

## 注意

- 目前数据保存在当前浏览器本地
- 如果你换手机、换浏览器、清空浏览器数据，记录不会自动同步
- 旧版本按分钟换算的数据会自动迁移为课时数
