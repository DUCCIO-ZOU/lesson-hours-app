# 课时达人

一个适合手机使用的课时记录小网页。

## 已有功能

- 新增上课记录
- 选择课时类型：阶梯课时 / 分成课时
- 自动把上课分钟数换算成课时
- 查看总记录数、总课时、本月课时
- 单独统计阶梯课时、分成课时
- 单独统计本月阶梯课时、本月分成课时
- 按月份筛选
- 按老师/课程搜索
- 按课时类型筛选
- 导出 CSV
- 数据保存在浏览器本地

## 怎么打开

### 电脑上
直接双击 `index.html` 即可。

### 手机上
要想手机随时打开，推荐把整个 `lesson-hours-app` 文件夹部署到静态网站：

- Cloudflare Pages
- Netlify
- Vercel
- GitHub Pages

部署后会得到一个网址，手机浏览器打开就能用。

## 注意

- 目前数据保存在当前浏览器本地
- 如果你换手机、换浏览器、清空浏览器数据，记录不会自动同步
- 如果你想要“账号登录 + 云端同步”，下一版可以继续加
